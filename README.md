# <u>Todo List</u>

This is a simple todo list made in python. 

## How to run this program?

##### Modules used: 
- Tkinter
- SQLite3 

Install Modules: 
```bash
pip install tkinter sqlite3
```
Clone the repo:
```bash
git clone https://github.com/Haardispro/Todo-List
```
Change the directory:
```bash
cd Todo-List
```
Run the program: 
```bash
python3 main.py
```
