## How to run this program?

Pre-requisites: 

- Python 3

- Tkinter

Run this program after installing the dependencies.


**To run the program:** 

Clone the repo:
```bash
git clone https://github.com/Haardispro/Todo-List
```
Change the directory:
```bash
cd Todo-List
```
Run the program: 
```bash
python3 main.py
```
